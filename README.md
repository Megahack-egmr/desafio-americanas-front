## Development server

Run `ionic serve` for a dev server. Navigate to `http://localhost:8100/`. The app will automatically reload if you change any of the source files.

## Installation

Download the [nodejs](https://nodejs.org/en/download/).

```bash
npm install
ionic serve 
```
